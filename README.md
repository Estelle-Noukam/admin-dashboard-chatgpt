# Admin Dashboard

## Description

Application web de tableau de bord administratif développée avec Flask.

## Fonctionnalités

- Authentification
- Gestion des utilisateurs
- Gestion des rôles
- Consultation des statistiques
- Tableau de bord administrateur
- Gestion des données

## Technologies

- Python
- Flask
- SQLite
- HTML
- CSS
- Docker

## Installation

pip install -r requirements.txt

python app.py

## Docker

docker-compose up --build

## Accès

http://localhost:5003
